### Installation 
ehh. først
```bash
pip install -U openai-whisper
```
deretter
```bash
# om du bruker mac
brew install ffmpeg

# eller om du bruker windows
choco install ffmpeg
```

tilslutt:
```bash
# for å altså bruke openai sin api
pip install openai 
```

om du vil kjøre server, bare kjør dette på terminalen
```
flask --app main run
```
